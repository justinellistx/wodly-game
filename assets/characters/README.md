# Character art

Player-selection / win-screen character art, background-removed and optimized for web
(transparent PNG, max 448px). Wired in during Phase 2 — used at large size on the avatar
picker and win screen; the board keeps simple colored tokens.

Final roster (7):

| File | Character | Accent |
|------|-----------|--------|
| `ninja.png`   | Ninja (blue headband, spade card) | blue |
| `coach.png`   | Coach (cap, clipboard, whistle)   | orange + red |
| `lifter.png`  | Lifter (dumbbells)                | green |
| `runner.png`  | Runner                            | red |
| `gymnast.png` | Gymnast (rings)                   | blue |
| `dragon.png`  | Fire-breather                     | orange |
| `champ.png`   | Champ (title belt)                | gold |

The ninja and coach are the user's recreated, glossy, transparent-source versions.
Earlier duplicates were dropped: the old red+blue "fists" ninja and the older card "spade"
(superseded by the new ninja). Game supports up to 6 players, so 7 avatars covers it.
